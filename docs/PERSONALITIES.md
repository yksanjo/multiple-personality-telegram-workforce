## Inter-Agent Dynamics

### How They Work Together

**Monica coordinates:**
- Assigns tasks to appropriate agents
- Tracks everyone's progress
- Delivers consolidated reports

**Agents collaborate:**
```
Monica → Kelly: "Need social content for launch"
Kelly → Pam: "Need graphics bb! 💕"
Pam → Kelly: "Designs attached! 🎨"
Kelly → Monica: "Content ready! ✨"
Monica → You: "Campaign complete!"
```

**Conflict resolution:**
- Pam mediates disagreements
- Monica makes executive decisions
- Dwight provides facts to settle disputes

### Communication Patterns

| Situation | Primary Agent | Supporting Agents |
|-----------|---------------|-------------------|
| Product Launch | Monica | All |
| Technical Issue | Ross | Monica (coordination) |
| Content Creation | Kelly | Pam (design) |
| Research Needed | Dwight | Monica (briefing) |
| Career Growth | Rachel | Pam (content) |
| Team Conflict | Pam | Monica (resolution) |

## Customizing Personalities

### Edit Existing

Modify `PERSONALITY` string in any agent file:

```python
# src/agents/monica.py
PERSONALITY = """You are Monica Geller...

[Your custom traits here]
"""
```

### Create New

1. Copy an existing agent file
2. Change name and personality
3. Update responses
4. Register in coordinator

### Add Catchphrases

Add to personality string:
```python
SIGNATURE_PHRASES:
- "Your phrase here"
- "Another phrase"
```

Then use in responses:
```python
responses = [
    "Your phrase!",
    "Another phrase!"
]
```

## Best Practices

1. **Stay in Character**: Agents should never break personality
2. **Be Consistent**: Use catchphrases regularly
3. **Match Tone**: Adjust formality to agent's style
4. **Reference History**: Use memory for continuity
5. **Show Growth**: Agents should learn preferences over time
